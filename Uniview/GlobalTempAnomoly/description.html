Global Temperature Anomoly

This module presents the Global Temperature changes from 1880-2017 from the Goddard Institute for Space Studies. The Measure we use is the Combined Land-Surface Air and Sea-Surface Water Temperature Anomalies (Land-Ocean Temperature Index, LOTI). Deatail of which are here. Note the module content only appears between simulation times 1800-2017.

The module will appear as "Global Temperature Anomoly" in the object tree, under Solar System/Planets/Earth.
